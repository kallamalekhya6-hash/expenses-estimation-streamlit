# Expenses Estimation App

This is a Streamlit-based web application to estimate and visualize monthly expenses.

## Features
- Category-wise expense input
- Automatic total & remaining balance calculation
- Expense distribution visualization
- Clean and user-friendly interface

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## How to Run
```bash
pip install -r requirements.txt
python -m streamlit run app.py
